Methods

(inner) GET /api/estadisticas/obtener/:idUsuario(req, res) → {Object|number|number|number}

Obtiene las estadísticas de un usuario (amigo) a partir de su ID. Las estadísticas se calculan utilizando los datos de las tablas "Juega" y "Partida".

req Object Objeto de solicitud HTTP.

params.idUsuario number ID del usuario para el cual se desean obtener estadísticas.
res Object Objeto de respuesta HTTP.
